# React To-Do List Application

This is a simple To-Do List application built using **React** and **Vite**.  
The app allows users to add, edit, delete, and mark tasks as completed.

The project is created using **Vite**, as required.

---

## Features

- Add new to-do items
- Mark tasks as completed
- Edit existing to-do items
- Delete to-do items
- Clean and user-friendly UI
- Built using functional components, state, and props

---

## Technologies Used

- React (Functional Components)
- Vite
- JavaScript (ES6)
- CSS
- Git & GitHub
